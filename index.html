Features

Our high-end authentication and encryption features will make your apps secure with just couple lines of code.

Authentication

Authenticate your app customers and sign operations using a strong multi-factor signature.

Secure Vault

Encrypt and securely store sensitive data or authentication related data on the mobile device.

E2E Encryption

Protect communication between your app and server with extra layer of end-to-end encryption

Strong Authorization

Sign sensitive transactions using a strong multi-factor symmetric key signature. HMAC‑SHA256

Fingerprint Authentication

Support biometric authentication with Touch ID on iOS or fingerprint authentication on Android.

Document Signatures

Sign documents and contracts with a private key using strong asymmetric signature. ECDSA‑P256r1

Jailbreak / Root Detection

Detect dangerous operating system modifications that can break your mobile application security.

Secure Push Notifications

AES-encrypt push notifications using a symmetric cipher with an ad-hoc generated key. AES‑128

Mobile OTP

Generate one-time passwords to authorize transactions and replace SMS messages in desktop and web applications.

QR Token

Make authentication and transaction signing convenient and fast with the help of QR codes.

Secure Tunnel

Encrypt communication, QR codes or documents so that only the right device can read them. AES‑128

Secure Storage

Store encrypted documents in a secure vault on a mobile device for fast access on slow networks.

Widgets

Show information on iOS or Android widgets to improve user experience without compromising security.

Wearables

Securely extend your mobile application with support for Apple Watch or Android Wear devices.

Secure Activation

Pair mobile devices with a user in a secure way to prevent the theft of the secret cryptographic keys.
